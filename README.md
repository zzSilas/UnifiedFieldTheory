# 统一场论 (Unified Field Theory)

这是一个关于张祥前统一场论的介绍网站和资源库。

## 项目内容

### 📄 文档
- **3统一场论.pdf** - 张祥前的完整统一场论著作，包含详细的理论推导和物理模型

### 🌐 网站

在 `website/` 目录中包含一个现代化的网站，用于介绍统一场论的核心概念。

#### 网站特性

- **科学极简主义设计** - 采用深蓝色和亮青色配色方案，营造宇宙深空的感觉
- **响应式设计** - 完美适配桌面、平板和移动设备
- **交互式内容** - 平滑的动画和过渡效果
- **详细的理论介绍** - 包括核心观点、物理模型、公式方程等

- **框架**: React 19 + TypeScript

- **路由**: Wouter

## 统一场论核心观点

### 1. 宇宙基本构成
宇宙由物体和空间构成，别的都不存在。物质和空间是宇宙的两个基本元素。

### 2. 空间螺旋运动
空间以螺旋状光速向外发散运动。这种运动是产生各种物理场和力的根本原因。

### 3. 四力统一
电磁力、引力、核力和弱力可以统一在一个数学方程中表达。

## 关键公式

### 基础关系式
```
E = m/(C-V)
```
物质与能量的基本关系，其中E为能量，m为质量，C为光速，V为速度。

### 动量方程
```
B = V × E /c²
```
动量与电场、磁场的关系。

### 场强方程
```
∂B/∂t = A×E/c²
```
磁场随时间的变化率与电场的关系。

## 本地开发

## 设计理念

网站采用**科学极简主义**设计风格，具有以下特点：

- **深邃与优雅** - 深蓝色背景搭配亮青色和紫色强调色
- **几何与秩序** - 使用几何图形和螺旋元素作为装饰
- **信息层级清晰** - 通过大小、颜色、间距建立明确的视觉层级
- **动态与静态平衡** - 适度使用动画和过渡效果

## 色彩方案

- **主色** - 深蓝色 (#0F172A) - 代表宇宙深空
- **辅助色** - 亮青色 (#00D9FF) - 代表能量和光
- **强调色** - 紫色 (#9D4EDD) - 代表神秘和理论深度
- **文本色** - 白色 (#FFFFFF) 和浅灰色 (#E0E0E0)

## 排版系统

- **标题字体** - Poppins (Bold 700) - 现代、几何感强
- **副标题字体** - Poppins (SemiBold 600)
- **正文字体** - Inter (Regular 400) - 高可读性
- **代码字体** - Fira Code (Regular 400) - 技术感
